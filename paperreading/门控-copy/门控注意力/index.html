本文是对semi-supervised learning(SSL)的一段简单串记和总结，用于加深对弱监督学习的理解，只进行了一些简单思想、模型的学习，主要参考了《An Overview of Deep Semi-Supervised Learning》这篇综述，对其中提到的方法进行了推敲和进一步的学习。

1.弱监督学习的介绍
#

1.1 什么是SSL
#

介于全监督和无监督学习之间，SSL是一种期望利用少量标记数据和大量未标记数据来进行网络性能提升的方法，该方法基于一些假设，并且基于这些假设，有许多不同类型的方法被提出，这将在后面进行具体阐述。

1.3 SSL方法的前提假定
#

所有方法都需要一个理论假设的前提，SSL也不例外，其基于三个主要假设：平滑性假定（The Smoothness Assumption）、聚类假定（The Cluster Assumption）和流形假定（The Manifold Assumption）。

具体来说：

**The Smoothness Assumption：**这个假定认为，相似的样本在特征空间中通常具有相似的标签。换句话说，如果两个样本在特征空间中很接近，它们很可能属于同一个类别。反之亦然；如果两个点被低密度区域分开，则输出必须彼此远离。这个假设对半监督学习至关重要，因为它允许模型在无标签样本上进行推断，并根据有标签样本的信息来预测无标签样本的标签。这个假设对于分类任务非常有帮助，但对于回归就基本没用了。

**The Cluster Assumption：**如果数据点属于同一聚类，则它们很可能属于相同的类别。在特定情况下的平滑性假定中，我们假设输入数据点形成聚类，每个聚类对应于输出类别的一个。聚类假定也可以被视为低密度分离假设：决策边界应该位于低密度区域。这两个假设之间的关系很容易理解，如果给定的决策边界位于高密度区域，它可能会将一个聚类划分为两个不同的类别，导致来自不同类别的样本属于同一聚类，这违反了聚类假定。在这种情况下，我们可以限制模型在一些小扰动下对无标签数据的预测保持一致，将决策边界推向低密度区域。总之，聚类假定强调了聚类和类别之间的关联性，它认为属于同一聚类的数据点很可能属于相同的类别。这个假设在半监督学习中有助于指导算法利用无标签数据来提高模型性能。

**The Manifold Assumption：**数据通常位于低维流形空间中，而这个流形空间的维度远远低于原始特征空间的维度。这个假设表明，虽然数据可能是高维的，但它们在一个较低维度的流形上更容易分布。因此，半监督学习方法通常试图在这个低维流形上建模数据分布。

2. Consistency Regularization（一致性正则化）
#

这种技术主要用于加强未标记数据的利用，确保训练出的模型符合聚类假设（cluster assumption），即学习到的决策边界应该位于低密度区域。

一致性正则化的基本思想是：对于一个未标记的样本，如果对它施加一个现实的扰动，其预测结果不应该发生显著变化。这是基于聚类假设，认为具有不同标签的数据点通过低密度区域分隔开，因此一个样本在受到扰动后转换到另一个类别的可能性很小

更正式地说，一致性正则化倾向于选择那些对相似数据点给出一致预测的函数 $f_{\theta} $。这意味着，相比于仅在输入空间的零维数据点上最小化分类成本，正则化模型会在每个数据点周围的流形上最小化成本，将决策边界推离未标记数据点，并在数据所在的流形上实现平滑化。

具体来说，对于一个未标记的数据点$ x \in D_u $ 及其扰动版本 $\hat{x}u $，目标是最小化两个输出之间的距离 $d(f{\theta}(x), f_{\theta}(\hat{x}))$ 。常见的距离测量方法包括均方误差（MSE）、Kullback-Leibler 散度（KL）和 Jensen-Shannon 散度（JS）。对于以类别概率分布形式的两个输出 $f_{\theta}(x) 和 f_{\theta}(\hat{x})$，以及$ m = \frac{1}{2} (f_{\theta}(x) + f_{\theta}(\hat{x})) $，可以如下计算这些测量：

  • 均方误差（MSE）: $$ d_{MSE}(f_{\theta}(x), f_{\theta}(\hat{x})) = \frac{1}{C} \sum_{k=1}^{C} (f_{\theta}(x)k - f{\theta}(\hat{x})_k)^2 $$

  • KL 散度: $$ d_{KL}(f_{\theta}(x), f_{\theta}(\hat{x})) = \frac{1}{C} \sum_{k=1}^{C} f_{\theta}(x)k \log \frac{f{\theta}(x)k}{f{\theta}(\hat{x})_k} $$

  • JS 散度:

$$ d_{JS}(f_{\theta}(x), f_{\theta}(\hat{x})) = \frac{1}{2} d_{KL}(f_{\theta}(x), m) + \frac{1}{2} d_{KL}(f_{\theta}(\hat{x}), m) $$

此外，也可以对$ x $ 的两个扰动版本$ \hat{x}_1 $和$ \hat{x}_2 $ 实施一致性。

2.1 Ladder Networks
#

Ladder Networks 是一种最简单的SSL方法，核心思路为：如果需要网络对噪声不敏感，那不如训练的时候就直接加上噪声，然后把噪声项放到损失函数里面去。

网络结构
#

  • 两个编码器（Encoders）：Ladder Network 包含两个编码器，一个处理干净的输入（clean encoder），另一个处理被噪声扰动的输入（noisy encoder）。

  • 一个解码器（Denoising Decoder）：用于从被噪声扰动的编码器输出中重建干净的输入和隐藏激活。

工作原理
#

在每次训练迭代中，输入 $x$ 同时被传递给两个编码器。在被噪声扰动的编码器中，在每层的批量归一化之后注入高斯噪声。 这样，网络产生两个输出：一个基于干净输入的预测 $y$ 和一个基于被噪声扰动激活的预测 $\tilde{y}$。 输出 $\tilde{y}$ 接着被送入解码器，以重建未被扰动的输入和干净的隐藏激活。

公式解释
#

无监督训练损失 $L_u$： $$ L_u = \frac{1}{|D|} \sum_{x \in D} \sum_{l=0}^{L} \lambda_l d_{MSE}(z^{(l)}, \hat{z}^{(l)}) $$ 其中，$|D|$ 是数据集中样本的数量，$L$ 是网络层的总数。$\lambda_l$ 是每层在总损失中的权重。$d_{MSE}$ 是均方误差函数，$z^{(l)}$ 和 $\hat{z}^{(l)}$ 分别是干净编码器和解码器在第 $l$ 层的激活。

有监督训练损失：

如果输入数据点 $x$ 有标签 $y$，则添加有监督的交叉熵损失 $H(\tilde{y}, t)$ 到无监督损失 $L_u$ 中，得到最终的总损失 $L$： $$ L = L_u + L_s = L_u + \frac{1}{|D_l|} \sum_{x,t \in D_l} H(\tilde{y}, t) $$ 其中，$|D_l|$ 是有标签数据点的数量，$H(\tilde{y}, t)$ 是预测 $\tilde{y}$ 和真实标签 $t$ 之间的交叉熵损失。

特点和应用
#

这种编码结构可以在卷积神经网络中很好的应用，通常在卷积神经网络的最后一层是全连接层，正好可以通过这种方式来代替，而前面的特征提取部分则不用改动。

该网络的计算成本相对较高，大约是普通训练迭代的三倍。

2.2 Pi-Model
#

Π-Model 特点
#

  • 简化架构：移除了 Ladder Networks 的nosi encoder，仅使用一个网络对所有输入进行处理。
  • 利用神经网络的随机性：在神经网络中加入dropout层可以很好的防止过拟合，因为这能减少各个神经元之间的相互依赖性。加入了dropout层之后，在每一次minibatch迭代中都随机的Unable一些神经元，在多次迭代的训练中还能起到集成学习的作用，这样能大大的提高网络预测的准确性。因此dropout也是一个正则化的好方法，那么对于无监督学习而言，同时，对数据进行增强（对于图像数据而言，可以有仿射变换或者裁剪）也是一个好方法。那么Π-Model 就是这两个方法的一个应用。

工作原理
#

对于任何给定的输入 $ x $，目标是减少两次前向传播中 $ f_{\theta} $ 对 $ x $ 的预测之间的距离。

具体实现：假设两次输出分别为 $ \tilde{y}_1 $ 和 $ \tilde{y}_2 $，由于预测函数的随机性（例如，使用 dropout 作为噪声源），这两个输出将会有所不同，目标是使得对于两者的预测保持一致。

训练过程和损失函数
#

无监督损失：如果输入 $ x $ 是未标记的数据点，则计算两个预测之间的均方误差（MSE）损失： $$ L = w \frac{1}{|D_u|} \sum_{x \in D_u} d_{MSE}(\tilde{y}1, \tilde{y}2) $$ 其中，$ |D_u| $ 是未标记数据集的大小，$ w $ 是一个权重函数，从 0 开始，经过一定数量的训练周期（例如，训练时间的 20%）增加到最大的权重 $ \lambda $（例如，30）。（原因：在训练初期时，因为模型很不稳定，因此无监督学习的误差$L = w \frac{1}{|D_u|} \sum{x \in D_u} d{MSE}(\tilde{y}_1, \tilde{y}_2)$

不是很有用)

有监督损失：如果 $ x $ 是有标签的数据点，还需要计算交叉熵损失： $$ L = w \frac{1}{|D_u|} \sum_{x \in D_u} d_{MSE}(\tilde{y}_1, \tilde{y}2) + \frac{1}{|D_l|} \sum{x, y \in D_l} H(y, f(x)) $$ 其中，$ |D_l| $ 是有标签数据集的大小，$ H(y, f(x)) $ 是预测 $ f(x) $ 和真实标签 $ y $ 之间的交叉熵损失。

特点
#

Π-Model 利用数据增强和随即失活方法增快了网络的训练速度，但仍然有问题：网络每次只考虑了单个mini batch的输入，而没有考虑所有数据的信息。从采样的角度来看，采样得到的数据集是对真实分布的一种采样，从一般情况来看，采样频率至少要大于等于奈奎斯特率才能还原真实分布，但多数情况我们是没有办法达成的，而minibatch更是对采样得到的数据集的一种估计，他与真实的数据分布的差距自然会更大。而在minibatch的训练中，只有多个minibatch的共同训练，相互影响，才能使得对真实数据的分布估计的更加准确。因此，单单考虑一个mini batch是不够的，所以说，对于无监督损失的衡量，就诞生了时间集成方法。

2.3 Temporal Ensembling
#

具体实现
#

  • 指数移动平均更新： $$ y_{ema} = \alpha y_{ema} + (1 - \alpha) \tilde{y} $$ 其中，$ \alpha $ 就是一个动量项，$ \alpha $ 越小，之前的信息对网络的影响就越小，当$ \alpha $ =0时，时间集成模型退化为Π-Model 。

    从直观上看，每一个的更新都是对之前的数据的值与当前值的加权平均，这样模型就很好的记住了之前的输出就能相互学习。

    做个比喻，就是温故而知新，当我们在复习备考时，一次minibatch就相当于做一套套题，梯度下降就相当于订正这张试卷，但是对于每一次新的minibatch，我们就不再复习之前的错误，因此时间久了我们就会忘掉前面做的卷子，这样效率就很差。指数平滑法就是利用了这种“复习”的思路，当前的卷子要订正，之前的错题也要整理。

    当然，光这样是不够的。在这里我们很容易能关注到一个问题：在初始化的时候，$y_{ema}$=0，但是此时输出$y$不为0，在初始几次迭代中，这样计算出来的MSE值会很大，同时因为网络没有训练好，公式更新如下：

  • 训练开始时的偏差校正： $$ y_{ema} = \frac{\alpha y_{ema} + (1 - \alpha) \tilde{y}}{1 - \alpha^t} $$ 类似于 Adam 优化器中的偏差校正，这样可以在训练开始时校正目标 $ y_{ema} $ 的初始噪声。

    在开始训练的时候，时间集成模型会退化至pi-model。这样的训练方法也有两个好处：一个是只需要一次前向传播，就可以计算出最后的结果，这大大的减少了计算时间（在深层网络中，一次前向传播的时间开销是很大的），另外该通过指数加权平均法也很好的防止了模型的不一致性。但是这样也有缺点：这需要有大量的内存来保存所有训练实力的预测集，对于大数据集的情况来看，这样对训练的要求也是很高的。

损失计算
#

  • 损失计算方式与 Π-Model 相同，但有两个重要优点：
    1. 训练加速：由于只需进行单次前向传播即可获得当前预测 $ \tilde{y} $，同时维护每个训练样本的标签预测的 EMA，并对与这些目标不一致的预测进行惩罚。
    2. 目标更稳定：训练过程中目标的稳定性得到提高，从而获得更好的结果。

缺点
#

  • 对于大型数据集和密集任务（例如语义分割），Temporal Ensembling 需要大量内存来保存所有训练样本的预测聚合，这可能会导致相当大的内存消耗。

Pi-model和时间集成模型都采取了集成学习的思想。然后也采取了teacher-student的训练思路，不过是自己当老师，自己当学生，也就是有监察机制，能比较好的解决抽样与分布的问题。但是，这也造成了一些问题：由于加权的原因，假设α值设的很大，那么新的新学到的信息需要完全被模型考虑到，需要经过很长的时间。而α值设的很小，之前的信息又不会被考虑的那么充分，那么导致一次的误差只会在一次minibatch训练中得到更新。

而且这个模型也有问题，因为在每一次训练中，都是“自己教自己”，假设在某一次训练中，无监督的损失的权重超过了有监督的损失，那模型就会被阻止学习新的信息，比如预测相同的目标。转而只把精力投入在无监督损失的优化当中。因此，对于模型预测的目标还是需要进行优化，方法如下：

仔细的选择噪声，而不是仅仅选择加性或乘性噪声。 仔细的选择导师模型，这个导师模型要对生成的学习目标负责，而不是简单的用学生模型来“自己教自己”。 为了实现这些目标，Mean teachers就出现了。

该论文是平均教师模型（MT）在医学影像分类方向的一个应用，在MT的基础上增加了a novel sample relation consistency (SRC) paradigm，也就是多增加了一个Loss：Lsrc

具体实现就是，除了分类头得到模型的输出外，在分类头前也进行输出并且进行reshape，从B,C,H,W变成B,C * H * W得到特征A，对特征A和A的转置进行矩阵乘法得到最后的特征sample relation matrix：R，shape为B * B。计算学生模型和教师模型R的MSE从而得到Lsrc损失。具体实现代码如下：

def relation_mse_loss(activations, ema_activations):
	assert activations.size() == ema_activations.size()
    
    activations = torch.reshape(activations, (activations.shape[0], -1))
    ema_activations = torch.reshape(ema_activations, (ema_activations.shape[0], -1))

    similarity = activations.mm(activations.t())
    norm = torch.reshape(torch.norm(similarity, 2, 1), (-1, 1))
    norm_similarity = similarity / norm

    ema_similarity = ema_activations.mm(ema_activations.t())
    ema_norm = torch.reshape(torch.norm(ema_similarity, 2, 1), (-1, 1))
    ema_norm_similarity = ema_similarity / ema_norm

    similarity_mse_loss = (norm_similarity-ema_norm_similarity)**2
    return similarity_mse_loss

其余的计算与损失函数和MT模型一样，有有监督的交叉熵损失，也有对于所有数据（有标签和无标签）的一致性损失，通过MSE实现。
