HashMap
#

创建 HashMap

Map<Integer, Integer> hashMap = new HashMap<>();

在 Java 中，HashMap<K, V> 是无序的，经常需要按 Key 或 Value 进行排序。

  1. 按照key排序

    • 使用 TreeMap 直接排序
     Map<Integer, Integer> sortedMap = new TreeMap<>(hashMap);
    • 使用 Comparator 和 List.sort()
    // 将 HashMap 转换为 List
List<Map.Entry<Integer, String>> list = new ArrayList<>(hashMap.entrySet());

// 按 Key 升序排序
list.sort(Comparator.comparingInt(Map.Entry::getKey));
    • 按 Key 降序
    list.sort(Comparator.comparingInt(Map.Entry::getKey).reversed());

  2. 按照Value排序

    HashMap 默认按 Key 查找，可以使用 Comparator 按 Value 排序。

    • 按 Value 升序
    // 转换为 List
List<Map.Entry<String, Integer>> list = new ArrayList<>(hashMap.entrySet());

// 按 Value 升序排序
list.sort(Comparator.comparingInt(Map.Entry::getValue));
    • 按 Value 降序
    list.sort(Comparator.comparingInt(Map.Entry::getValue).reversed());

  3. HashMap 结合 PriorityQueue（堆排序）

    • 用 PriorityQueue 找出 Value 最大的前 K 个元素
    // 小顶堆，按 Value 升序排序
PriorityQueue<Map.Entry<String, Integer>> minHeap =
   new PriorityQueue<>(Comparator.comparingInt(Map.Entry::getValue));

// 遍历 HashMap，加入堆
for (Map.Entry<String, Integer> entry : freqMap.entrySet()) {
   minHeap.offer(entry);
   if (minHeap.size() > k) {
       minHeap.poll(); // 只保留最大的 K 个
   }
}

// 输出前 K 个最大值
while (!minHeap.isEmpty()) {
   System.out.println(minHeap.poll());
}
    • 用 PriorityQueue 构造「大顶堆」按 Value 降序排列
    PriorityQueue<Map.Entry<String, Integer>> maxHeap =
   new PriorityQueue<>(Collections.reverseOrder(Comparator.comparingInt(Map.Entry::getValue)));

maxHeap.addAll(freqMap.entrySet());

while (!maxHeap.isEmpty()) {
   System.out.println(maxHeap.poll());
}

循环结构
#

  • int为32位，long为64位，如果需要更大的位数则需要使用 BigInteger,它可以处理任意大小的整数.

    下面为一个示例代码，用于计算阶乘和求和。当n过大时，则需要使用BigInteger.

    import java.util.Scanner;
import java.math.BigInteger;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int n = scanner.nextInt();
        BigInteger total = BigInteger.ZERO; // 使用 BigInteger 存储总和
        while (n > 0) {
            total = total.add(fac(n)); // 使用 add 方法相加
            n--;
        }
        System.out.println(total);
    }

    // 使用 BigInteger 计算阶乘
    public static BigInteger fac(int n) {
        if (n == 1) return BigInteger.ONE;
        else return fac(n - 1).multiply(BigInteger.valueOf(n)); // 使用 multiply 方法相乘
    }
}

    sum[i + 1] = sum[i] + num[i];
}
long minValue = Long.MAX_VALUE;

    long windowSum = sum[i] - sum[i - m];
    minValue = Math.min(minValue, windowSum);
}

//只维护 windowSum 变量
int windowSum = 0;
for (int i = 0; i < m; i++) {
    windowSum += num[i];
}

int minSum = windowSum;

for (int i = m; i < n; i++) {
    windowSum += num[i] - num[i - m];
    minSum = Math.min(minSum, windowSum);
}

  • 判断是否为质数

    //法1 
boolean isPrime(int num) {
    if (num < 2) return false;
    for (int i = 2; i * i <= num; i++) { // 只检查 `2 ~ sqrt(N)`
        if (num % i == 0) return false;
    }
    return true;
}

//法2 埃拉托色尼筛法
private static void sieve(int limit) {
    // 1. 初始化所有数为“可能是质数”
    for (int i = 0; i <= limit; i++) isPrime[i] = true;

    // 2. 0 和 1 不是质数
    isPrime[0] = isPrime[1] = false;

    // 3. 遍历所有数，并筛除它们的倍数
    for (int i = 2; i * i <= limit; i++) { // 只检查到 sqrt(limit)
        if (isPrime[i]) { // i 是质数，则筛去 i 的倍数
            for (int j = i * i; j <= limit; j += i) { // 从 i² 开始筛
                isPrime[j] = false;
            }
        }
    }
}

    法2更加高效

方法 时间复杂度 适用范围
暴力检查 O(N) O(N) 单个数
检查 2 ~ sqrt(N) O(√N) 单个数
埃拉托色尼筛法 O(N log log N) 批量判断（N ≤ 10^7）
  • 公式：所有子集元素之和 $=(s_1 + s_2 + \cdots + s_n) \times 2^{n-1}=\bigl(\sum_{i=1}^{n} s_i\bigr) \times 2^{n-1}.$
